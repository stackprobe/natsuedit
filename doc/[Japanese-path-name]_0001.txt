===============================================================================

　夏Edit

===============================================================================


■ソフトの概要

　シンプルな動画ファイルのエディタです。

　短い動画の一部をぼかしたいという目的だけで作ったので、大した機能はありません。
　ニッチな需要に応えつつ機能を増やしていけたらなぁと思います。


　★当ソフトは ffmpeg を使用します。(外部コマンドとして呼び出します)
　　当ソフト使用前に ffmpeg を導入していただく必要があります。

　★数分〜数十分程度の短い動画の編集を想定しています。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。

　.NET Framework 4 を使用します。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■ffmpeg導入方法

　1. ブラウザから https://ffmpeg.org/ を開いて下さい。

　2. ページ左側のメニューの Download をクリックして下さい。

　3. Get the packages の Windows のロゴにカーソルを合わせ、
　   Windows Packages の Windows Builds をクリックして下さい。

　4. Version の 3.2.4 を選択して下さい。(3.2.4 が無い場合は、最新の安定版を選択)

　5. Architecture の 32-bit または 64-bit を「導入環境に合わせて」選択して下さい。
　   よくわからない場合は 32-bit を選択して下さい。

　6. Linking の Shared を選択して下さい。(Static でも OK です)

　7. Download FFmpeg をクリックして .zip ファイルをダウンロードして下さい。

　8. ダウンロードした .zip を任意の場所に展開して下さい。
　   展開先のパスは後で使用するので、覚えておいて下さい。


　★この手順は 2017/3/13 に作成しました。

　★開発・テスト環境では ffmpeg-3.2.4-win64-shared.zip を使用しています。

　★開発・テスト環境では C:\app\ffmpeg-3.2.4-win64-shared に展開しました。


■起動方法

　natsuedit.exe を実行して下さい。

　natsuedit.exe のアイコン又はショートカットに動画ファイルをドラッグ・アンド・ドロップして、
　動画ファイルを開くこともできます。


■使い方

　アプリ(A) / ファイルを開く(O) から動画ファイルを開きます。
　ウィンドウに動画ファイルをドラッグ・アンド・ドロップしても開けます。

　ウィンドウ下部にあるシークバーで映像の位置を変えられます。
　シークバーの左側にある [<], [>] ボタンで１フレームずつシークすることもできます。

　動画の情報、現在のシーク位置、選択状態などはウィンドウ下部のステータス領域に表示されます。

　必要に応じて「画面選択」「時間選択」を行い、
　「エフェクト」から任意の加工を行って下さい。

　アプリ(A) / 名前を付けて保存(A) から編集した動画を保存できます。


■画面選択

　映像の一部または全体の矩形領域を選択できます。
　エフェクトによっては画面選択が必要です。

　ウィンドウ中央に表示されている映像をドラッグすることによって、
　左ボタンを押した位置から、放した位置までの矩形領域を選択することができます。

　画面選択(S) / 数値入力(I) から選択範囲を細かく修正できます。

　映像をクリックすると、選択状態を解除できます。


■時間選択

　映像の一部または全体の範囲（何フレーム目から何フレーム目まで）を選択できます。
　エフェクトによっては時間選択が必要です。

　シークバーで任意の位置にシークしてからシークバーを右クリックし、
　ここから・ここまで(S) を選択すると、選択領域の一端に指定されます。
　更に、シークバーで任意の位置にシークしてからシークバーを右クリックし、
　ここから・ここまで(S) を選択すると、選択領域のもう一端に指定され、時間選択が完了します。


■エフェクト

　●切り捨てる

　　時間選択された範囲を動画から削除します。

　●ぼかし１

　　画面選択・時間選択された範囲にぼかしを入れます。

　●ぼかし２

　　画面選択・時間選択された範囲にぼかしを「強めに」入れます。

　●枠外切り捨て

　　画面選択された範囲「外」を映像から削除します。

　●字幕を入れる

　　時間選択された範囲に字幕を入れます。


■補足

　動画ファイルは、映像を .png に、音声を .wav に変換して編集を行います。
　そのため動画ファイルから変換するとき及び動画ファイルに戻すときに多かれ少なかれ映像・音声が「劣化」します。
　同じファイルを何度も「開いて、保存」すると、著しく劣化するのでご注意下さい。

　保存した動画ファイルはコーデックが変わってしまうため WMP, qt などで再生出来なくなるようです。
　（手前の環境でですが VLC, Google Chrome では再生出来るようです）

　ログファイル natsuedit.log, natsuedit0.log は、合計およそ 2 MB を超えないように管理（自動的に削除）されます。
